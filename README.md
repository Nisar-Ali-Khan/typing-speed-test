# type.test ⌨️

A clean, beautiful typing speed test built with pure HTML, CSS, and JavaScript.

## Features
- Live WPM, Accuracy, Timer & Streak tracking
- 3 modes — Words, Sentences, Code
- 3 time limits — 30s, 60s, 2 minutes
- Smooth word-by-word scrolling
- Grade system — S / A / B / C
- Dark mode support
- No frameworks, no dependencies

## How to Use
Just open `typing-speed-test.html` in any browser and start typing!
